# Branches

## New branch

```
git checkout main
git checkout -b new_branch
git push -u origin new_branch
```

## Switch branch

```
git checkout branch_name
```

## Delete branch

```
git checkout main
git branch -D old_branch
git push origin --delete old_branch
git branch -r
```
